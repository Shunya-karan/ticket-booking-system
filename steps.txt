🚀 Step 1: Create Project Folder
Open your PC → Create a folder:
ticket-booking-system
Right-click → Open with VS Code



🧱 Step 2: Create Frontend (React)
Inside VS Code Terminal:
npx create-react-app frontend
Wait → It will generate code.
Now go into frontend folder:
cd frontend
Run to test:
npm start
React app will open in browser → Good ✅


🖥️ Step 3: Create Backend Folder
In VS Code Terminal (go back to root):
cd ..
mkdir backend
cd backend
npm init -y
Install backend packages:
npm install express cors mysql2 sequelize bcrypt jsonwebtoken
